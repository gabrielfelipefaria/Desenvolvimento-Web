# 🚀 Projeto Geek - Front-End (HTML, CSS e JavaScript)

Este projeto foi desenvolvido como parte de atividades práticas de desenvolvimento web, com foco em HTML, CSS e JavaScript.

O objetivo é simular diferentes funcionalidades inspiradas no universo geek, incluindo interações, estilização moderna e lógica básica de programação.

---

## 📌 Funcionalidades

✔ Perfil de personagem geek  
✔ Card estilo jogo (herói/vilão)  
✔ Lista de episódios com destaque  
✔ Botão interativo (modo Jedi/Sith)  
✔ Sistema de pontuação (XP)  
✔ Quiz geek com pontuação  
✔ Alternância de tema (claro/escuro)  
✔ Galeria interativa de personagens  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
